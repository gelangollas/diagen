Этот модуль и тот модули.

Главной программой объединяются следующие модули: Settings, View, Image Processor, 
Worker, Contour Processor и Fractal Calculator.
Модуль Settings отвечает за координирование работы настроек системы.
Модуль View отвечает за представление пользователю визуальных
данных работы системы.
Модуль Image Processor отвечает за обработку изображений, включает в себя модули: Binarizer, для морфологической обработки.
Модуль Binarizer отвечает за перевод изображения в бинарный формат.
Модуль Morphology Processor за морфологическую обработку
изображения.
Модуль Worker координирует работу системы и является связующим
звеном между представлением и внутренним устройством системы.
Модуль Contour Processor отвечает за обработку контуров, объединяет модули Contour Processor, Contour Enhancer.
Модуль Contour Searcher отвечает за поиск контуров на обработанном
изображении.
Модуль Contour Enhancer ответственен за приведение контуров к
финальному виду .
Модуль Fractal Calculator отвечает за расчет фрактальной размерности, является точкой входя для модулей Box Counter и Least Squares Calculator.
Модуль Box Counter отвечает за отработку алгоритма Box Counting.
Модуль Least Squares Calculator отвечает за расчет углового
коэффициента линии регрессии методом наименьших квадратов.
